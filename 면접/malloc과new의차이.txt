c++면접에 나올 예상 질문.

malloc() == 함수
new == 연산자
가장 큰 차이는 생성자의 유무
malloc()은 시스템 함수로서 함수 안에서 메모리를 할당하지만
new는 연산자로 바로 메모리를 할당하는게 아니라 생성자를 호출하여 메모리를 할당한다.
생성자로 초기화하기 때문에 메모리를 할당할 때 생성 시 초기화가 가능하다는 장점이 있다.

malloc(), calloc(), realloc()의 차이점
realloc()의 대표적 문제는 메모리 할당이 실패할 경우 null이 반환되기 때문에
기존의 메모리가 할당되어 있는 포인터를 잃어버리는 것이다.
->기존의 메모리 주소를 저장하고 실패 시 복구하는 프로세스가 함께 정의 되어야 한다.

생성 시 초기화가 가능하다는데 그건 어떻게 하는 것인가?
초기화 리스트와 대입의 차이
초기화 리스트 : 생성 시 초기화와 생성 후 초기화 

Car() {
바퀴 = 4;
엔진 = 1;
기름 = 0;
} 은 생성 후 초기화이고

Car() : 바퀴(4), 엔진(1), 기름(0) {
} 은 생성 시 초기화이다.
효울은 생성 시 초기화가 더 좋다. 생성 시 초기화는 한 흐름에 모든 것이 끝나는 반면
생성 후 초기화는 생성과 초기화 2가지 흐름으로 처리되기 때문이다.
